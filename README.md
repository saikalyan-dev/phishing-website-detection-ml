# phishing-website-detection-ml
Phishing Website Detection using Machine Learning
This project detects phishing websites using machine learning algorithms.

## Technologies Used
- Python
- Scikit-learn
- Pandas
- NumPy

## Features
- Detects malicious URLs
- Machine learning classification
- Data preprocessing

## How to Run

1. Install dependencies

pip install -r requirements.txt

2. Run the model

python phishing_model.py
